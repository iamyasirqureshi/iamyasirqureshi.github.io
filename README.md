## ✨ Features

- ⚡ **Pure HTML/CSS/JS** — zero dependencies, blazing fast
- 🌙 **Dark glassmorphism UI** with smooth animations
- 📱 **Fully responsive** — works on mobile, tablet & desktop
- 🖱️ **Custom cursor** with interactive ring effect

## 📁 Project Structure

```
yasirqureshi.dev/
│
├── index.html          # Main HTML file (single-page portfolio)
├── style.css           # All styles — layout, animations, glassmorphism
├── script.js           # Interactivity — cursor, navbar, animations
├── sitemap.xml         # SEO sitemap for Google indexing
```

